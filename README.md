# LuxRand-Prelim2
True random number generation via hardware entropy. This project leverages a "floating" A0 analogue pin to capture ambient electromagnetic noise. By sampling high-frequency atmospheric fluctuations and processing them through a Von Neumann whitener, it converts unpredictable analogue signals into a reliable, non-deterministic digital seed.
